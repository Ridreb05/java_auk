# Java Programming Lab (AUK)

This repository contains a collection of Java programs developed for the laboratory sessions of the B.Tech Computer Science and Engineering (CSE) curriculum at Amity University, Kolkata (AUK). 

The programs cover fundamental to advanced object-oriented programming (OOP) concepts in Java, including data structures, file handling, multithreading, and basic GUI development.

## 📂 Repository Structure

* **`/Basic-Programs`**: Syntax, data types, control flow, and basic logic.
* **`/OOP-Concepts`**: Classes, objects, inheritance, polymorphism, and encapsulation.
* **`/Data-Structures`**: Arrays, linked lists, stacks, and queues implemented in Java.
* **`/Exception-Handling`**: Try-catch blocks, custom exceptions, and error management.
* **`/Multithreading`**: Thread creation, synchronization, and lifecycle management.
* **`/File-Handling`**: Reading from and writing to files using Java I/O streams.
